# Ecommerce Dental Logic - Frontend M2 🦷💻

Proyecto desarrollado por **Nataly Santana** para el módulo de Frontend. 
Se trata de una plataforma de soluciones digitales para laboratorios dentales, integrando la precisión técnica con el desarrollo en Python.

## 🚀 Requisitos Técnicos Cumplidos
* **HTML5 Semántico:** Uso de `header`, `nav`, `main`, `section` y `footer`.
* **Bootstrap 5.3 (vía CDN):** * Implementación de Grid (Container, Row, Col).
    * Componentes: Navbar, Cards, Buttons y Badges dinámicos.
    * Utilidades de espaciado y tipografía.
* **JavaScript Vanilla:** * Manejo de eventos (Click).
    * Manipulación del DOM (Actualización de carrito).
    * Uso de Arrays y Objetos para la lógica de productos.
* **Git/GitHub:** Repositorio público con historial de commits descriptivos.
